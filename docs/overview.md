# 课程大纲

面向非技术同学的编程入门课程，共 6 次，每周 1 次，每次 1 小时。

目标：帮已经在用 AI 写代码的同学补上底层认知——理解代码为什么能跑、出了问题怎么定位、项目变复杂了怎么管理。

---

## 第 1 课：全局地图 — 代码到底在哪里运行

- 浏览器输入 URL 到页面展示，中间经历了什么
- 前端文件 vs 后端文件 vs 配置文件的区别
- Chrome DevTools 基本用法：Elements、Network、Console

## 第 2 课：Git — 代码的存档与协作

- 为什么需要版本管理
- 核心操作：add → commit → push → 回退
- 在 GitHub 上查看提交记录

## 第 3 课：读代码 — 看懂 AI 写了什么

- 读代码的方法：找入口，顺着数据流走
- React 项目的结构：入口 → 路由 → 页面 → 组件
- 读代码时会遇到的 JS 核心概念速览

## 第 4 课：React 心智模型 — 组件、状态、渲染

- 组件是什么：输入 props，输出 UI
- 状态是什么：数据变了，页面自动更新
- 渲染流程：状态改变 → 组件重新执行 → 页面更新

## 第 5 课：后端 — 数据存在哪、API 是什么

- 前端和后端的分工
- API：前后端之间的约定
- 动手写一个最简单的后端接口

## 第 6 课：调试 + 部署 — 出了问题怎么办，做完了怎么上线

- 报错信息怎么读，常见错误模式
- 用 Vercel 一键部署，把项目分享给别人用
