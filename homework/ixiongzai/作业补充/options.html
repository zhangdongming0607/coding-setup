如何获取配置信息？

  1. 获取 API 密钥：
    ① 访问 Notion Integrations 页面
    ② 点击 "New integration" 创建新集成
    ③ 填写名称（如"Chrome 划词收藏"），类型选择 "Internal"
    ④ 创建后，找到 "Internal Integration Token"，点击复制按钮复制这串以 secret_ 开头的长字符串
  2. 创建 Notion 数据库：
    ① 在 Notion 中创建一个新的数据库（Database）
    ② ⚠️ 重要：数据库必须包含以下四个属性（列），否则扩展无法正常工作！
    点击数据库右侧的 + 号，逐个添加以下属性：
    • 标题（Title 类型，通常已存在，如果没有请添加）
    • URL（URL 类型，列名必须完全一致，区分大小写）
    • 收藏时间（Date 类型，列名必须完全一致）
    • 标签（Multi-select 类型，列名可以是"标签"、"Tag"或"tags"三者之一）
    ③ 确保列名完全匹配（URL 区分大小写），否则扩展无法正常保存数据
  3. 获取数据库 ID：
    ① 打开你创建的数据库页面
    ② 查看浏览器地址栏，你会看到类似这样的 URL：
    https://www.notion.so/xiongzai/2dcff72112918075a98ef30ea7c53970?v=...
    ③ 直接复制整个 URL 粘贴到右侧输入框，系统会自动提取 32 位数据库 ID
    或者手动复制 URL 中最后一个斜杠 / 和问号 ? 之间的 32 个字符
  4. 连接数据库（重要！）：
    ① 在数据库页面，点击右上角的 "..." 菜单
    ② 选择 "Connections"（连接）
    ③ 点击 "Add connections"，勾选你刚才创建的集成
    ④ 完成连接
