# 作业提交说明

## 提交步骤

1. 打开终端，进入课程仓库目录
2. 把作业文件放到 `homework/你的文件夹/lesson-X/` 下面
3. 执行以下命令：

```bash
git pull
git add homework/你的文件夹/
git commit -m "第 X 课作业"
git push
```

## 注意事项

- 每次作业放到自己文件夹下对应的 `lesson-X/` 里
- 不要改别人的文件夹
- 文件名随意，能看出内容就行
- push 之前先 `git pull`，避免冲突
