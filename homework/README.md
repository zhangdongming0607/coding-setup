# 作业提交说明

## 提交步骤

1. 打开终端，进入课程仓库目录
2. 把作业文件放到 `homework/你的文件夹/` 下面
3. 执行以下命令：

```bash
git add homework/你的文件夹/
git commit -m "第 X 课作业"
git push
```

## 注意事项

- 每次作业放到自己的文件夹里，不要改别人的文件夹
- 文件名随意，能看出是哪次作业就行（比如 `lesson-1-map.md`）
- push 之前先 `git pull` 一下，避免冲突
