# 第 1 课作业

## 任务 1：看看你的页面发了哪些请求

1. 打开 Demo 项目（或你自己的 vibe coding 项目）
2. 按 `Cmd+Option+I`（Mac）/ `F12`（Windows）打开 DevTools
3. 切到 **Network** 面板
4. 刷新页面
5. 数一下一共发了几个网络请求
6. 找到一个 API 请求（类型是 Fetch/XHR 的那种），点开看 Response 里返回了什么数据
7. 截图放到作业里

## 任务 2：触发一个错误请求

1. 在聊天框里输入 `错误` 两个字，点发送
2. 看 Network 面板里多了什么？状态码是多少？颜色有什么变化？
3. 再输入一条正常消息（比如"你好"），对比两次请求的区别
4. 用几句话写下你观察到的不同

## 任务 3：画一张"项目地图"

1. 打开你自己做的 vibe coding 项目（或 Demo 项目）
2. 列出所有重要的文件和文件夹
3. 每个文件用一句话说明它是干嘛的
4. 标注哪些是"前端文件"、哪些是"后端文件"、哪些是"配置文件"

格式不限——文本列表、手绘拍照、用任何你顺手的工具都行。
